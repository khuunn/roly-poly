`uv run pytest`를 실행하고 결과를 분석하세요.

## 출력 형식

| 항목 | 결과 |
|------|------|
| 전체 테스트 | N건 |
| 통과 | N건 |
| 실패 | N건 |
| 스킵 | N건 |
| 소요 시간 | N.Ns |

실패한 테스트가 있으면:
1. 실패 원인을 분석하세요
2. 관련 소스 코드를 읽고 원인을 파악하세요
3. 수정 방안을 제시하세요 (사용자 동의 후 수정)
